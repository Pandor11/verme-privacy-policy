Политика конфиденциальности мобильного приложения
ООО «ВЕРМЕ»

Дата публикации:

1. Общие положения

Настоящая политика конфиденциальности (далее — Политика) описывает порядок обработки персональных данных при использовании мобильного приложения ООО «ВЕРМЕ» (далее — Приложение).

ООО «ВЕРМЕ» уважает право пользователей на конфиденциальность и обязуется соблюдать действующее законодательство Российской Федерации в области персональных данных, включая Федеральный закон №152-ФЗ «О персональных данных».

2. Назначение и функциональность приложения

Приложение предназначено исключительно для удобного доступа к личному кабинету пользователей через сканирование QR-кода. После сканирования пользователь автоматически переходит по ссылке, открывающейся во встроенном WebView-компоненте, который загружает официальный сайт компании.

3. Обработка персональных данных

  • Приложение не осуществляет самостоятельный сбор, обработку или хранение персональных данных.
  • Вся обработка персональных данных происходит на стороне веб-сайта, открываемого внутри приложения.
  • Приложение не передаёт данные третьим лицам и не использует сторонние аналитические или рекламные SDK.
  • Приложение может сохранять технические данные (домен, cookies), необходимые для корректной работы авторизации внутри WebView.

4. Согласие пользователя

Устанавливая и используя Приложение, пользователь подтверждает своё согласие с настоящей Политикой.

5. Политика конфиденциальности сайта

Поскольку Приложение отображает контент официального сайта ООО «ВЕРМЕ», к обработке персональных данных также применяется Политика конфиденциальности сайта, размещённая по следующей ссылке:

https://verme.ru/policy

6. Изменения политики

Настоящая Политика может быть обновлена в случае изменений в законодательстве или в функциональности Приложения. Актуальная версия всегда доступна по данной ссылке.

7. Контакты

По вопросам, связанным с обработкой персональных данных, вы можете связаться с ООО «ВЕРМЕ» по электронной почте: support@verme.ru.
